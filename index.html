Fake Visitor Counter

現在、人が訪問しています。


概要

よくサイトで表示される『現在〇〇人がこのページを見ています。』というのを偽装するためのJavaScriptです。


仕様

初期値は35〜40までのランダム、数値の変化は5秒ごとに

  • -2: 2%
  • -1: 15%
  • +1: 15%
  • +2: 2%

の確率で変化します。

変化の上限は50、加減は25としています。

Cookieに値を保存することで、リロードしてもバレないようにしています。


使い方

まず、HTMLにcount_of_visitorsというタグを作ります。

次に、fetch_count_of_visitors.jsをscriptタグで導入すれば完了です。

設定は簡単にカスタマイズすることができます。

fetch_count_of_visitors.jsの中にある変数configを書き換えることで、設定を変更できます。


        var config = {
          // 初期値の範囲（整数）
          initRange: {
            min: 35,
            max: 40
          },
          // 変化の範囲（整数）
          variationRange: {
            min: 25,
            max: 50
          },
          // 変化の確率（合計が100未満になる整数）
          percentage: {
            '-2': 2,
            '-1': 15,
            '+1': 15,
            '+2': 2
          }
        };
